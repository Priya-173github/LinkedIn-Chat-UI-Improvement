01 — Problem

What's broken
with LinkedIn's chat

LinkedIn's messaging interface has accumulated usability debt. Misleading badge counts, flat message hierarchy, missing read receipts, and a cluttered compose bar all create unnecessary friction.

Before Current issues
✕

Misleading badge count

Counts total messages, not unique senders. One person sending 3 messages shows "3" — creating false urgency.

✕

Flat message hierarchy

Every message gets a timestamp and full avatar, making threads hard to scan and visually noisy.

✕

No read receipts

Senders can't tell if a message has been seen, leading to repeat follow-ups and uncertainty.

✕

Cluttered compose area

All actions are crowded at the same visual level with no clear hierarchy between tools.

After Design solutions
✓

Conversation-count badge

Badge now reflects unique users with unread messages — one conversation = one count, regardless of message volume.

✓

Grouped message threads

Consecutive messages from the same person cluster visually. One avatar and one timestamp per group.

✓

Read receipts added

Green checkmark confirms when the recipient has read your last message, reducing ambiguity.

✓

Structured input bar

Media actions separated from emoji with a divider. Compose field and send button are clearly prominent.

The logic error

Notification badges should represent the number of people who need your attention, not content volume. WhatsApp, iMessage, and Telegram all do this correctly — LinkedIn is the outlier.

05 — Design Principles

Four guiding principles

01
Conversation-first

The interface recedes so the conversation breathes. No chrome competes with the content of messages.

02
Mental model fidelity

Every UI signal maps to how users actually think. Badges count people, not messages. Ticks communicate stages, not just delivery.

03
Scannable hierarchy

Grouped messages reduce repetitive avatars and timestamps. Users find what they need in one pass, not after parsing every element.

04
Formal but warm

Professional without being cold. Light type weights, generous spacing, and a teal success palette create approachability within structure.
